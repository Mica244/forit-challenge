# ForIT challenge 2026
Aplicacion de lista de tareas desarrollada con React, Node.js, Express y SQLite3.

## Tecnologias utilizadas

- **Frontend:** React + vite + Bootstrap
- **Backend:** Node.js + Express
- **Base de datos:** SQLite3 (better-sqlite3)
- **Estilos:** Bootstrap + CSS

## Funcionalidades:

- Crear, editar y eliminar tareas
- Marcar tareas como completadas
- Filtrar por todas/pendientes/completadas
- Busqueda por titulo
- Persistencia de datos con SQLite3
- Validacion de formularios

## Instalacion y uso:
### Backend
```bash
cd backend
npm install
npm run dev
```
### Frontend:
```bash
cd frontend
npm install
npm run dev
```
### Variables de entorno

**backend/.env**
 PORT=3000
**frontend/.env**
VITE_API_URL=http://localhost:3000
